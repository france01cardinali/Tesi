# Configuratore tirocinio

## Avvio progetto

Apri due terminali nella cartella del progetto:

```bash
cd configuratore_tirocinio
```

### 1. Backend HTTPS

Nel primo terminale:

```bash
npm run backend:https:local
```

Il backend parte su:

```text
https://localhost:3001
```

### 2. Frontend HTTPS

Nel secondo terminale:

```bash
npm run start:https:local
```

Il frontend parte su:

```text
https://localhost:4200
```

Apri nel browser:

```text
https://localhost:4200
```

## Build

Per verificare che il progetto compili:

```bash
npm run build
```
